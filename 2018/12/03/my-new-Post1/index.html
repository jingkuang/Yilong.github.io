WIndows系统如何制作博客

十二月 03, 2018

安装前提：

Nodejs：在官网 http://nodejs.cn/ 进行下载，默认安装或选择路径安装
Git：在官网 https://git-scm.com/ 进行下载，安装的时候选择能在cmd中运行
文本解释器：
Visual studio code（导师推荐）或者sublime text（个人推荐）
安装hexo
Hexo:是一个快速、简洁且高效的博客框架。Hexo使用Markdown（或其他渲染引擎）解析文章，在几
秒内，即可利用靓丽的主题生成静态网页
安装成功后通过WIN+R在cmd中运行命令：npm install -g hexo-cli 安装hexo
看到hexo-cli@1.1.0

安装成功后新建文件夹blog
通过路径表里面打开cmd,输入hexo启动
输入hexo init . (.代表当前目录/路径)

将文件夹拖到Sublime text程序中

文件中，“_config.yml”是整个网站的总配置文件
1.进入文件需要修改“title（标题可写）、description（描述可写）、author（作者=用户名）、language（语言：中文ZH-CH）以及url（博客地址）等”（根据自己需求）

2.登录github，
点击右上角“+”号，选择“New repository”创建新库


注意：Owner(所有者)和Repository name（存储卡名称）一致后缀名为github.io

完成创建
3.选择“create new file”写入文件
博客地址：https://jingkuang.github.io/
下载静态文件并且完成本地访问
在文档中打开cmd窗口
4.下载静态文件：
执行命令：hexo generate

目的：下载静态文件到本地
5.本地访问：
执行命令：hexo server

目的：开启服务器

重点同步本地文件至GitHub

  1. 打开GitHubDesktop并登录
  2. 选择“clone a repository”
  3. 选择数据库，设置存放文件的位置，点击“clone”
  4. 把写好的文件放入上一步设置的路径中（注:必须是静态文件），打开GitHub Desktop
    选择要上传的文件，写完摘要，描述，点击“commit to master”
  5. 同步操作：
    点击，即可上传完成
