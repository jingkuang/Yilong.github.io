博客如何更换主题

十二月 03, 2018

将主题下载下来

关于下载方法：
1.可以复制链接 在命令行使用git clone 链接进行下载
2.手动下载

将下载下来的主题放在Blog/themes中

进入Blog/themes通过路径进入cmd命令行

输入 hexo clean 清除缓存 （有兴趣的可以查看少什么文件）

hexo generate 下载缓存
hexo server 重新启动

主题完成后

将E:\Blog\public重新复制到C:\Users\DELL\Documents\GitHub\jingkuang.github.io
完成上传

文章需要在根据提示如：hexo new “my new Post”

系统给出具体位置如：E：\blog\source_postes\my-new-Post.md
进入文件

文件形式为

title: my new Post
date: 2018-12-03 14:50:44
tags:
---
