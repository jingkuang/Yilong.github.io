安装前准备

下载安装SSH（我使用的Termius，就按照Termius讲解）
将虚拟机地址和账号密码输入到Edit Host 点击右上角SAVE保存即可
进入Termius，开始使用

1.将httpd-2.4.37文件放到/root（我是通过Cyberduck上传文件）


2.解压httpd-2.4.37.tar.gz: tar -xzvf httpd-2.4.37.tar.gz

开始正式安装

1.进入httpd-2.4.37文件夹cd httpd-2.4.37

2.安装前看INSTALL或者README，可以看到安装步骤

建议可以用终端重新创建一个命令行端口

3.按照步骤执行命令
使用命令：./configure –prefix=/usr/local/apache2。这时会出现错误

error:APR not found:（错误：没有APR）

4.查看APR所需的文件：yum search APR

使用yum install apr-devel apr-util apr -y 下载
5.再次使用./configure –prefix=/usr/local/apache2 再次出现错误 error: APR-util not found

6.下载所缺失的程序
继续使用yum search APR-util

找到缺失文件使用yum install apr-util apr-util-devel -y
继续使用./configure –prefix=/usr/local/apache2命令会发现出现错误

error:in “/root/httpd-2.4.37” error:C compiler cannot create executables这是提醒我们需要安装开发者工具集，使用
yum groupinstall “development tools” -y

7.继续使用安装命令./configure –prefix=/usr/local/apache2再次出现错误：error: pcre-config for libpcre not found.

8.重复yum命令yum search pcre

继续使用yum install pcre-devel pcre -y
9.再次执行./configure –prefix=/usr/local/apache2

10.执行成功

安装成功

这时，我们已经执行成功了，但是还没有安装所以我们需要进行安装。
使用命令make && make install
完成安装

如何运行Apache程序

使用命令

/usr/local/apache2/bin/apachectl start

上图中所出现的文字并不会影响程序的运行。可以正常使用静态网页
